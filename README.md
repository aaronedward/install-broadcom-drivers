# BCM4360 WiFi Driver Installation Script

This script automates the installation of the BCM4360 driver on Ubuntu 26.04 systems. 

## Prerequisites

- Ubuntu 26.04 (or compatible Debian-based system)
- Internet connection for package downloads

## Usage

1. Make script executable: `chmod +x install-bcm4360-wifi.sh`
2. Run as root: `sudo ./install-bcm4360-wifi.sh`

## Notes

- The script will automatically reboot the system after completion
- This script is specifically for BCM4360 chipsets

## Compatibility

- Intended for use on an A1466 MacBook Air running Ubuntu 26.04 with Kernel 7.0+
- May work on other Debian-based distributions
